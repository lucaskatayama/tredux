

# Tredux
A Trello implementation using React/Redux.


# Demo
[Demo here](http://github.lucaskatayama.com/tredux/)

# Installation
I'm using [Yarn](yarnpkg.com) instead of NPM.

```
$ cd /path/to/repo
$ yarn
$ yarn run start
```

# Build

```
$ yarn run build
```
